# SkillSwap App

A mobile app built with React Native and Firebase. Users post skills, earn credits, and request skills from others. This project was built for an academic assignment focused on mobile development, authentication, CRUD operations, and navigation.

## Features
• User registration and login  
• Firebase Authentication with persistence  
• Firestore database for skills and requests  
• Live skill feed  
• Post a skill  
• Track your own requests  
• Profile and logout  
• Bottom tab navigation  
• Clean UI with consistent spacing and layout  

## Tech Stack
• React Native  
• Expo  
• Firebase Auth  
• Cloud Firestore  
• React Navigation
